# React + Vite

## Instalacion

Para poder ejecutar este proyecto, es necesario instalar las dependencias de node. Para ello, se debe ejecutar los siguientes comandos:
    ```
    cd /OrderNow-React/
    npm install -g npm@latest
    npm install
    ```
