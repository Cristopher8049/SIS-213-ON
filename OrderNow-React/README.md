# React + Vite + Tailwind

## Instalacion

Para poder ejecutar este proyecto es necesario ejecutar los siguientes comandos:

    cd /OrderNow-React/

    npm install -g npm@latest
    npm install
    npm install react-router-dom

    npm install tailwindcss @tailwindcss/vite
    
    npm run dev
